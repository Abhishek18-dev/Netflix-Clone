# 🎬 Netflix Clone (HTML & CSS)

This project is a **Netflix Clone UI** built using only **HTML and CSS**.
It was created while learning web development fundamentals from CodeWithHarry's Sigma Web Development Playlist.

---

## 🚀 Project Goal

The main goal of this project was to:

* Strengthen **HTML structure**
* Master **CSS styling & layouts**
* Understand **Flexbox & positioning**
* Practice **real-world UI cloning**

---

## 🖥️ Features

* 🎨 Clean and modern Netflix-inspired UI
* 📱 Responsive design
* 🧱 Structured layout using Flexbox
* 🎬 Hero section with background image
* 🔘 Styled buttons

---

## 🛠️ Tech Stack

## ▶️ How to Use

Follow these steps to run the project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/netflix-clone-html-css.git
   ```

2. **Open the project folder**

   ```bash
   cd netflix-clone-html-css
   ```

3. **Run the project**

   * Simply open `index.html` in your browser
   * OR right-click → Open with Live Server (in VS Code)
